# Student Management System

A console-based Java application that performs CRUD operations using JDBC and MySQL.

## Features
- Add student
- View all students
- Update student details
- Delete student
- Menu-driven console interface

## Technologies Used
- Java (Core Java)
- JDBC
- MySQL

## Project Structure
- Student.java – Model class
- DBConnection.java – Handles database connection
- StudentDAO.java – Contains CRUD operations
- MainApp.java – Menu-driven main application

## How to Run
1. Create MySQL database and table
2. Update DB credentials in DBConnection.java
3. Run MainApp.java
